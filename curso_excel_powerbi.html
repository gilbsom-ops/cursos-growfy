Módulo 1 · Excel Básico

Interface e Fundamentos do Excel

⏱ ~70 min📖 Teoria + Prática

Por que Excel é essencial no mercado financeiro?

O Excel é a ferramenta mais utilizada no mercado financeiro mundial. Analistas, gestores e assessores usam diariamente para modelagem financeira, controle de carteiras, relatórios e dashboards. Dominar o Excel é o primeiro passo para se destacar profissionalmente.

Interface do Excel — Elementos principais

ElementoFunção

Caixa de NomeExibe o endereço da célula (ex: A1, B3)
Guias (Abas)Planilhas dentro do arquivo (Sheet1, Sheet2...)
Faixa de Opções (Ribbon)Menu com todas as funcionalidades organizadas por categoria
Área de TrabalhoGrade de células (linhas × colunas)

Referências de Células

  • Relativa (A1): Muda quando a fórmula é copiada para outra célula.
  • Absoluta ($A$1): Não muda ao copiar — use F4 para fixar.
  • Mista ($A1 ou A$1): Fixa só a coluna ou só a linha.
=A1+B1 → relativa | =$A$1+$B$1 → absoluta | =$A1+B$1 → mista
Use $ para fixar referências que não devem mudar ao copiar a fórmula
💡 Atalho essencial
Pressione F4 ao editar uma referência de célula para alternar entre relativa, absoluta e mista. Economiza muito tempo!

Formatação de Dados

  • Número: Para valores genéricos com casas decimais
  • Moeda: R$ com 2 casas decimais
  • Porcentagem: Multiplica por 100 e adiciona %
  • Data: Reconhece formatos dd/mm/aaaa

Módulo 2 · Excel Básico

Fórmulas Essenciais do Excel

⏱ ~90 min📖 Teoria + Prática

As fórmulas mais usadas no mercado financeiro

Conhecer as fórmulas certas é o que separa quem usa o Excel como calculadora de quem o usa como ferramenta profissional. As fórmulas abaixo são as mais cobradas em processos seletivos do setor financeiro.

Fórmulas Matemáticas Básicas

=SOMA(A1:A10) → soma do intervalo A1 até A10
Também: =SOMA(A1;B1;C1) para células não contíguas
=MÉDIA(B1:B12) → média aritmética do intervalo
=MÍNIMO(intervalo) e =MÁXIMO(intervalo) para menor e maior valor
=CONT.NÚM(A1:A50) → conta células com números | =CONT.VALORES → conta não-vazias
=CONT.SE(intervalo;"critério") → conta com condição

Função SE — Lógica Condicional

=SE(teste_lógico; valor_se_verdadeiro; valor_se_falso)
Exemplo: =SE(B2>=7;"Aprovado";"Reprovado")

SE aninhado (múltiplas condições):

=SE(A1>=90;"Excelente";SE(A1>=70;"Aprovado";SE(A1>=50;"Recuperação";"Reprovado")))
Resultado verificado: 95→Excelente | 75→Aprovado | 55→Recuperação | 30→Reprovado ✅

Função PROCV — Busca Vertical

=PROCV(valor_procurado; tabela; núm_coluna; [correspondência_exata])
Use 0 ou FALSO para correspondência exata. Use 1 ou VERDADEIRO para aproximada.
💡 Exemplo prático PROCV
Tabela de clientes na coluna A (CPF) e B (Nome). Para buscar o nome pelo CPF:
=PROCV(E2;A:B;2;0) → busca o valor de E2 na coluna A e retorna o valor da coluna 2 (B).

Funções de Texto

FunçãoUsoExemplo
=CONCATENAR() ou &Une textos=A1&" "&B1 → "João Silva"
=MAIÚSCULA()Converte para maiúsculas=MAIÚSCULA("growfy") → "GROWFY"
=NÚM.CARACT()Conta caracteres=NÚM.CARACT("Excel") → 5
=ARRUMAR()Remove espaços extrasÚtil ao importar dados

Módulo 3 · Excel Intermediário

Fórmulas Avançadas — ÍNDICE, CORRESP e SOMASES

⏱ ~90 min📖 Teoria + Prática

Por que ir além do PROCV?

O PROCV tem limitações: só busca da esquerda para direita e exige que a coluna de busca seja a primeira. As funções ÍNDICE+CORRESP resolvem isso e são muito mais flexíveis e poderosas.

Função CORRESP

=CORRESP(valor_procurado; intervalo; [tipo])
Retorna a POSIÇÃO do valor no intervalo. Tipo 0 = exata. Ex: =CORRESP("ETH";A1:A10;0) → retorna 3 se ETH estiver na 3ª linha

Função ÍNDICE

=ÍNDICE(intervalo; núm_linha; [núm_coluna])
Retorna o VALOR na posição indicada. Ex: =ÍNDICE(B1:B10;3) → retorna o valor na 3ª linha de B1:B10

ÍNDICE + CORRESP — A Combinação Poderosa

=ÍNDICE(coluna_retorno; CORRESP(valor_buscado; coluna_busca; 0))
Busca em qualquer direção, mais rápido que PROCV em bases grandes. O CORRESP encontra a posição e o ÍNDICE traz o valor.
💡 Exemplo: busca bidirecional
Buscar o preço de um ativo (coluna D) pelo código (coluna A), mesmo que a coluna de retorno esteja À ESQUERDA da busca:
=ÍNDICE(D:D;CORRESP(G2;A:A;0)) — impossível com PROCV!

SOMASES — Soma com Múltiplos Critérios

=SOMASES(intervalo_soma; intervalo_critério1; critério1; intervalo_critério2; critério2; ...)
Soma apenas as células que atendem a TODOS os critérios simultaneamente

Exemplo verificado: somar vendas do mês "Jan" e tipo "Vendas":

=SOMASES(C:C;A:A;"Jan";B:B;"Vendas") → R$ 1.000 ✅
Combina múltiplos filtros em uma única fórmula — essencial para relatórios financeiros

CONT.SES e MÉDIASES

=CONT.SES(intervalo1;critério1;intervalo2;critério2)
Conta registros com múltiplos critérios. =MÉDIASES calcula a média com critérios.

Funções de Data

=DIAÚTIL(data;dias)Data útil N dias após a data

Módulo 4 · Excel Intermediário

Tabelas Dinâmicas — Análise Rápida de Dados

⏱ ~80 min📖 Teoria + Prática

O que são Tabelas Dinâmicas?

Tabelas Dinâmicas (Pivot Tables) são uma das funcionalidades mais poderosas do Excel. Permitem resumir, analisar, explorar e apresentar grandes volumes de dados com poucos cliques — sem escrever uma única fórmula.

💡 Caso de uso real
Você tem 50.000 linhas de transações financeiras. Com uma Tabela Dinâmica, em 30 segundos você sabe o total por categoria, por mês, por filial — qualquer combinação imaginável.

Como criar uma Tabela Dinâmica

  1. Selecione qualquer célula dentro dos seus dados
  2. Vá em Inserir → Tabela Dinâmica
  3. Escolha onde colocar (nova planilha recomendado)
  4. No Painel de Campos, arraste campos para as áreas:
ÁreaFunçãoExemplo
FiltrosFiltro geral da tabelaAno
ColunasCabeçalho de colunasTrimestre
LinhasCabeçalho de linhasCategoria
ValoresO que calcularSoma de Receita

Configurações de Valor

Clique no campo de Valores e escolha a operação: Soma, Contagem, Média, Máximo, Mínimo, % do Total, % do Total Geral etc.

💡 Mostrar como %
Em "Configurações do Campo de Valor" → "Mostrar Valores Como" → "% do Total Geral". Transforma os valores em percentual automaticamente — ótimo para análise de participação.

Segmentações de Dados (Slicers)

Segmentações são filtros visuais interativos conectados à Tabela Dinâmica. Para inserir: Analisar → Inserir Segmentação de Dados. Clique nos botões para filtrar instantaneamente — ideal para dashboards.

Atualizar a Tabela Dinâmica

⚠️ Importante
Quando os dados de origem mudam, a Tabela Dinâmica NÃO se atualiza automaticamente. Clique com o botão direito na tabela → Atualizar (ou Ctrl+Alt+F5 para atualizar todas).

Gráfico Dinâmico

Diretamente de uma Tabela Dinâmica: Analisar → Gráfico Dinâmico. O gráfico se atualiza junto com a tabela e suporta as mesmas segmentações de dados.

✏️

Módulo 5 · Excel Intermediário

Gráficos Profissionais no Excel

⏱ ~70 min📖 Teoria + Prática

Gráficos como ferramenta de comunicação

No mercado financeiro, um gráfico bem construído pode ser mais persuasivo que uma tabela com 1.000 números. A escolha do tipo certo de gráfico e a formatação adequada são fundamentais para transmitir a mensagem correta.

Tipos de Gráfico e quando usar

TipoMelhor paraExemplo financeiro
Colunas / BarrasComparação entre categoriasReceita por produto, faturamento por filial
LinhasEvolução temporalCotação de ações, saldo ao longo do tempo
Pizza / RoscaParticipação no total (até 5 categorias)Composição de carteira, % por setor
ÁreaVolume acumulado no tempoPatrimônio acumulado, evolução de fundos
Dispersão (XY)Correlação entre variáveisRisco vs Retorno de ativos
Cascata (Waterfall)Composição de variaçõesDRE — do Faturamento ao Lucro

Boas práticas de visualização

  • Título claro e informativo: "Receita Mensal 2024" em vez de "Gráfico1"
  • Menos é mais: Remova grades desnecessárias, bordas e elementos que não agregam
  • Rótulos de dados: Adicione valores nos pontos mais importantes
  • Escala adequada: Nunca comece o eixo Y em valor diferente de zero sem boa razão
  • Cores consistentes: Use a paleta da empresa, não o padrão colorido do Excel
⚠️ Erros comuns em gráficos financeiros
  • Gráfico 3D: distorce a percepção visual das proporções — evite sempre
  • Pizza com muitas fatias: difícil de ler acima de 5-6 categorias
  • Eixo Y cortado: cria impressão de variações maiores que a realidade

Formatação Condicional

A formatação condicional aplica formatação automática com base em regras. Essencial para destacar alertas financeiros:

  • Escala de Cores: Verde (maior) → Vermelho (menor) para rentabilidades
  • Barras de Dados: Mini-gráfico dentro da célula proporcional ao valor
  • Conjunto de Ícones: Setas ↑ ↓ para indicar tendência
  • Realçar células: Pintar de vermelho valores negativos automaticamente
✏️

Módulo 6 · Excel Finanças

Funções Financeiras do Excel

⏱ ~90 min📖 Teoria + Prática

Excel como calculadora financeira profissional

O Excel possui funções financeiras nativas que substituem calculadoras HP e são amplamente usadas em modelagem financeira, análise de projetos e planejamento de investimentos.

PGTO — Calcular Parcela de Financiamento

=PGTO(taxa; nper; vp; [vf]; [tipo])
taxa = taxa por período | nper = nº de períodos | vp = valor presente (financiado)

Financiamento de R$ 50.000 em 60 meses a 1% ao mês:

=PGTO(1%;60;50000) = -R$ 1.112,22/mês ✅
O resultado é negativo pois representa saída de caixa. Use =ABS(PGTO(...)) para exibir positivo.

VP — Valor Presente

=VP(taxa; nper; pgto; [vf]; [tipo])
Quanto vale hoje uma série de pagamentos futuros

VP de 24 parcelas de R$ 500 a 1% ao mês:

=VP(1%;24;-500) = R$ 10.621,69 ✅
Quanto você pode financiar hoje pagando R$500/mês por 24 meses a 1% a.m.

VF — Valor Futuro

=VF(taxa; nper; pgto; [vp]; [tipo])
Quanto acumula investindo periodicamente

Investindo R$ 300/mês por 36 meses a 1% ao mês:

=VF(1%;36;-300) = R$ 12.923,06 ✅
Note o sinal negativo no pagamento — representa saída de caixa (investimento)

TAXA — Calcular a Taxa de Juros

=TAXA(nper; pgto; vp; [vf]; [tipo]; [estimativa])
Encontra a taxa implícita em um financiamento ou investimento

Financiamento de R$10.000 em 24 parcelas de R$500: taxa ≈ 1,5131% a.m. ✅

TIR — Taxa Interna de Retorno

=TIR(valores; [estimativa])
O primeiro valor deve ser negativo (investimento inicial). Os demais são os retornos.

Projeto: -R$10.000, retornos de R$3.000, R$4.000, R$4.000 e R$3.000:

=TIR({-10000;3000;4000;4000;3000}) ≈ 14,90% por período ✅

VPL — Valor Presente Líquido

=VPL(taxa; valores) + investimento_inicial
ATENÇÃO: =VPL() no Excel não inclui o t=0. Some o investimento inicial (negativo) separadamente!
✏️

Introdução ao Power BI

⏱ ~80 min📖 Teoria + Prática

O que é Power BI?

O Power BI é a ferramenta de Business Intelligence (BI) da Microsoft. Permite conectar diversas fontes de dados, transformá-los e criar dashboards e relatórios interativos que se atualizam automaticamente. É gratuito na versão Desktop e amplamente usado no mercado financeiro corporativo.

💡 Excel vs Power BI — Quando usar cada um?
Excel: Análises ad hoc, modelos financeiros, cálculos complexos, pequenas bases de dados (até ~1 milhão de linhas).
Power BI: Dashboards interativos, grandes volumes de dados (milhões de linhas), múltiplas fontes integradas, relatórios compartilhados para equipes.

Versões do Power BI

VersãoCustoUso
Power BI DesktopGratuitoCriação de relatórios no computador
Power BI Service (Online)Gratuito (limitado)Publicar e compartilhar relatórios
Power BI Pro~US$ 10/mêsCompartilhamento completo entre usuários
Power BI PremiumCorporativoGrande escala, IA avançada

Fluxo de trabalho no Power BI

  1. Conectar: Importar dados de Excel, CSV, SQL, API, Web etc.
  2. Transformar: Limpar e modelar dados no Power Query
  3. Modelar: Criar relacionamentos entre tabelas e medidas DAX
  4. Visualizar: Criar visuais (gráficos, tabelas, KPIs, mapas)
  5. Publicar: Compartilhar no Power BI Service

Módulo 8 · Power BI

Conectando e Transformando Dados — Power Query

⏱ ~80 min📖 Teoria + Prática

Power Query — O Motor de Transformação de Dados

O Power Query é o editor de transformação de dados do Power BI (e também do Excel moderno). É onde você "limpa" os dados antes de analisá-los. Cada transformação é registrada como um passo e pode ser reeditada ou reordenada.

Conectando Fontes de Dados

FonteComo conectar
Excel / CSVObter Dados → Excel / Texto/CSV
Banco de dados SQLObter Dados → SQL Server (ou outro)
Web (tabelas HTML)Obter Dados → Web → colar URL
SharePoint / OneDriveObter Dados → SharePoint
API RESTObter Dados → Web → URL da API

Transformações mais usadas no Power Query

  • Remover colunas desnecessárias: Clique com botão direito → Remover
  • Renomear colunas: Duplo clique no cabeçalho
  • Filtrar linhas: Seta no cabeçalho → Filtros
  • Alterar tipo de dado: Clicar no ícone à esquerda do cabeçalho
  • Dividir coluna: Por delimitador (ex: separar "Nome Sobrenome")
  • Mesclar consultas: Equivalente ao PROCV/JOIN entre tabelas
  • Acrescentar consultas: Empilha tabelas (ex: dados de meses diferentes)
💡 Unpivot — Despivotar colunas
Dados mensais em colunas (Jan, Fev, Mar...) precisam ser convertidos para linhas para análise no Power BI. Selecione as colunas de meses → Transformar → Despivotar Colunas. Isso cria duas colunas: "Atributo" (mês) e "Valor".

Medidas DAX — Introdução

DAX (Data Analysis Expressions) é a linguagem de fórmulas do Power BI. É usada para criar medidas calculadas que respondem aos filtros dos visuais dinamicamente.

Total Vendas = SUM(Vendas[Valor])
Medida básica: soma da coluna Valor da tabela Vendas
Ticket Médio = DIVIDE([Total Vendas]; COUNTROWS(Vendas))
DIVIDE é preferível a / pois trata divisão por zero elegantemente
Vendas YTD = TOTALYTD([Total Vendas]; Calendario[Data])
YTD = Year To Date (acumulado no ano) — requer tabela de calendário

Módulo 9 · Power BI

Dashboards Financeiros no Power BI

⏱ ~90 min📖 Teoria + Prática

O que faz um bom dashboard financeiro?

Um dashboard financeiro eficaz responde perguntas de negócio em segundos, sem necessidade de explicação. Os melhores dashboards seguem princípios claros de design e foco no usuário.

Visuais essenciais para finanças

VisualUso financeiro
Cartão (Card)KPIs principais: Receita Total, Lucro, Inadimplência
Gráfico de LinhasEvolução temporal: faturamento mês a mês
Gráfico de Colunas EmpilhadasComposição por categoria ao longo do tempo
Gráfico de CascataDRE: do Faturamento ao Lucro Líquido
Mapa CoropléticoDesempenho por estado/região
Tabela / MatrizDetalhamento com drill-down
SegmentaçãoFiltros interativos: período, produto, filial

KPIs — Indicadores-Chave de Performance

Um dashboard financeiro profissional sempre inclui KPIs no topo, visíveis de imediato. Os principais:

  • Receita Total: faturamento do período
  • Margem Bruta %: (Receita - CPV) / Receita
  • EBITDA: lucro antes de juros, impostos, depreciação e amortização
  • Inadimplência %: carteira vencida / carteira total
  • ROE: Retorno sobre Patrimônio Líquido
💡 Layout recomendado para dashboard financeiro
Linha 1 (topo): 4-5 Cartões de KPI com variação vs período anterior
Linha 2 (meio): Gráfico de linhas (evolução) + Gráfico de colunas (composição)
Linha 3 (base): Tabela de detalhamento + Mapa ou gráfico adicional
Lateral: Segmentações (Ano, Mês, Produto, Filial)

Interatividade no Power BI

Todos os visuais num relatório Power BI são interativos por padrão: clicar em um elemento filtra os demais automaticamente. Isso é chamado de cross-filtering.

💡 Drill-through
Permite navegar de um resumo para o detalhe. Ex: clicar em "Vendas São Paulo" no mapa leva a uma página com todos os detalhes de SP. Configure em: Campos → área "Filtros de Drill-through".

Medidas DAX para dashboards financeiros

Variação % vs Mês Anterior = DIVIDE([Total Vendas] - [Vendas Mês Ant.]; [Vendas Mês Ant.])
Compara o período atual com o anterior — essencial em qualquer dashboard financeiro
Margem % = DIVIDE([Lucro Bruto]; [Receita Total])
Formatar como % no painel de formatação do visual

Neste módulo final você vai integrar tudo: conexão de dados, Power Query, DAX e visualização. Vamos construir um dashboard de Demonstração de Resultados (DRE) completo, com comparativos mensais e indicadores de performance.

Estrutura do Projeto

EtapaFerramentaO que fazer
1. DadosExcelPlanilha com faturamento, custos e despesas por mês

Medidas DAX do Projeto

Receita Total = SUM(DRE[Receita])
Base de todas as outras medidas

CPV = Custo dos Produtos/Serviços Vendidos

Formatar como % — mostra eficiência operacional
EBITDA = [Lucro Bruto] - SUM(DRE[DespesasOperacionais])
Lucro operacional antes de juros, IR, depreciação
Receita vs Mês Ant = DIVIDE([Receita Total]-CALCULATE([Receita Total];DATEADD(Calendario[Data];-1;MONTH));CALCULATE([Receita Total];DATEADD(Calendario[Data];-1;MONTH)))
